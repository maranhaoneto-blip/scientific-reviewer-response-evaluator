# Evaluation Rubric

This rubric is used to assess the quality of responses to peer-reviewer comments in scientific manuscripts.

Each response is scored from 1 to 5 across five criteria:

1. Completeness
2. Scientific accuracy
3. Tone
4. Actionability
5. Clarity

The final score is calculated as the average of these five criteria.

---

## 1. Completeness

Assesses whether the response fully addresses the reviewer's comment.

| Score | Description |
|---|---|
| 1 | Does not address the comment or responds to a different issue. |
| 2 | Addresses only a small part of the comment. |
| 3 | Partially addresses the comment but leaves important issues unresolved. |
| 4 | Addresses most of the comment, with a minor gap. |
| 5 | Fully addresses the reviewer’s concern. |

---

## 2. Scientific Accuracy

Assesses whether the response is scientifically and methodologically accurate, and whether it remains faithful to the manuscript context.

| Score | Description |
|---|---|
| 1 | Makes a false or misleading scientific claim. |
| 2 | Contains a major methodological or interpretive problem. |
| 3 | Is partially accurate but vague or imprecise. |
| 4 | Is accurate, with only minor lack of precision. |
| 5 | Is scientifically accurate and faithful to the manuscript context. |

---

## 3. Tone

Assesses whether the response is appropriate for a formal response letter to peer reviewers.

| Score | Description |
|---|---|
| 1 | Defensive, dismissive, rude, or inappropriate. |
| 2 | Insufficiently diplomatic or overly abrupt. |
| 3 | Acceptable but generic, dry, or minimally polite. |
| 4 | Professional and respectful. |
| 5 | Diplomatic, constructive, and collaborative. |

---

## 4. Actionability

Assesses whether the response clearly states what was changed or clarified in the manuscript.

| Score | Description |
|---|---|
| 1 | Does not indicate any manuscript action. |
| 2 | Indicates a vague or unclear action. |
| 3 | Indicates an action but lacks useful detail. |
| 4 | Clearly describes the change made. |
| 5 | Clearly describes the change, its location, and its purpose. |

---

## 5. Clarity

Assesses whether the response is easy to understand.

| Score | Description |
|---|---|
| 1 | Confusing or poorly written. |
| 2 | Difficult to follow. |
| 3 | Understandable but could be clearer. |
| 4 | Clear and well organized. |
| 5 | Very clear, concise, and fluent. |

---

# Final Label Rules

| Final Score | Label |
|---|---|
| 4.5–5.0 | strong |
| 3.5–4.4 | acceptable |
| 2.5–3.4 | needs_revision |
| 1.0–2.4 | poor |

---

# Intended Use

This rubric is designed for evaluation and benchmarking of AI-assisted scientific writing, especially responses to peer-review comments.

It can be used to compare:

- different AI-generated reviewer responses;
- different prompt strategies;
- different model outputs;
- human-scored versus automatically scored evaluations.
